PhotoConnect Privacy Policy

Last Updated: July 2026

PhotoConnect is a photo printing application for compatible CS3 photo printers. This application does not require user registration or login.

Information We Collect

The application may collect the following information:

  • Printer serial number
  • Application version
  • Android OS version
  • Printer authorization status

This information is collected only for printer authorization, compatibility verification and troubleshooting.

Photos

Photos selected by the user remain on the device. They are never uploaded to our servers.

Third-party Services

The application uses:

  • Firebase Remote Config
  • Firebase Firestore

These services are used only to manage printer authorization and application configuration.

Advertising

This application contains no advertisements and does not use the Android Advertising ID.

Data Sharing

Collected information is never sold to third parties.

Data Deletion

If you would like any stored device-related data removed, please contact the developer.
